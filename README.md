# EDR Real Estate Mastery — Agent Training Platform

## Overview
A 12-week online training program for new California real estate licensees at EDR Realty Group. Built and maintained by the EDR training team.

## Platform URL
**https://edrbroker.github.io/EDR-real-estate-mastery**
